# SchiebePokerGame-Development
The project implements a Schiebepoker game using Kotlin and the BoardGameWork framework.

This project was developed as part of the course "Software Practice" at TU Dortmund.
The system design and architecture (including the class diagram) were created in a team of 9 students, 
while the full implementation of the code was completed independently by me.

How to run the game:
1. Download the repository and open the distribution folder.
2. Navigate to the bin folder.
3. Start the application by running projekt1.bat.
